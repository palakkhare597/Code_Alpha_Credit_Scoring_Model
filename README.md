# 💳 Credit Scoring Model

A Machine Learning project developed as part of the **CodeAlpha Machine Learning Internship**. This project predicts an individual's creditworthiness using historical financial data. The model is trained using **Decision Tree** and **Random Forest** algorithms and deployed with **Streamlit**.

---

## 📌 Project Overview

The Credit Scoring Model helps determine whether a customer is likely to be creditworthy based on financial information. This assists financial institutions in making better lending decisions.

---

## 🚀 Features

- Data preprocessing and cleaning
- Decision Tree Classifier
- Random Forest Classifier
- Model performance evaluation
- Accuracy, ROC-AUC Score
- Classification Report
- Confusion Matrix
- Feature Importance Visualization
- Interactive Streamlit Web Application
